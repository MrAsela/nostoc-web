About Nostoc Network

The Mycelium Principle

Nostoc is built on the biological model of mycelium networks - decentralized, resilient, and self-healing structures that facilitate communication and resource sharing without a central point of control.

Just as mycelium connects trees in a forest, Nostoc connects individuals in a secure, anonymous network where information spreads like spores through the digital ecosystem.

Core Features

  • Anonymous posting: No accounts, no tracking, no connection to your identity
  • End-to-end encryption: All content is encrypted before leaving your device
  • Decentralized storage: Content is distributed across multiple nodes in the network
  • Pinned spores: Important messages from network curators
  • Trusted admins: Verified network maintainers ensure content quality

Network Protocol

Nostoc uses a custom protocol combining:

  • Onion routing for anonymous communication
  • Blockchain-inspired timestamping for content verification
  • Peer-to-peer content distribution
  • Zero-knowledge proofs for admin authentication

The network is designed to be resistant to censorship and surveillance, with no single point of failure.
